  • 미디어쿼리란

    디바이스의 환경에 따라

    사전에 설정해둔 값에 맞춰서 반응형으로 홈페이지를 표현해 줄 수 있는

    코딩 기법

    반응형 웹 디자인(Responsive Web Design)의 핵심 요소

    미디어 유형은 장치의 종류를 나타내며, 예를 들어 스크린(screen), 프린터(printer), 스피커(speech) 등이 있습니다

  • 왜 사용하는가

    넓은 화면에서 보이던 레이아웃이 작은화면에

    맞지 않는 가독성과 구조의 복잡함을 해결하기위해

    각 사이즈별로 디자인을 변경할 수 있도록

    미디어쿼리를 사용이 가능하며

    상황에 맞게 다크 모드를 사용중인 사람이라면 다크모드로 표현도 가능하며

    홈페이지를 사용하는 사용자에게 조금더 사용성이 좋게 디자인된 페이지를 제공할 수 있다.

  • 유형 및 종류

    미디어 유형에는 all(모든 매체), print(프린터 기기), screen(컴퓨터, 태블릿, 스마트폰 등), speech(스크린 리더) 등이 있다.

    이 중 all, screen, print를 가장 많이 쓰이는데 생략이 가능해서 생략하셔도 되며. 생략할 경우 기본값은 all 이다.

    미디어 유형 설명
    all 모든 미디어 유형
    screen 컴퓨터 스크린(스마트폰 스크린 포함) - 가장 많이 사용
    print 인쇄 장치
    tv 음성과 영상이 동시 출력되는 TV
    aural 음성 합성 장치(주로 화면을 읽어 소리로 출력해주는 장치)
    braille 점자 표시 장치
    handheld 패드(pad)처럼 손에 들고 다니는 장치
    projection 프로젝터
    tty 디스플레이 기능이 제한된 장치(픽셀(px) 단위를 사용할 수 없음)
    embossed 점자 프린터

  • 작성방법

    미디어 쿼리는 CSS의 @media 규칙을 사용하여 정의해야 한다.

    @media only screen and (max-width:800px){ 변경할 CSS 코딩 }

    위 코드 해석

    화면 너비가 800px 이하인 경우에만 적용되는 스타일

    해당 쿼리 내용으로는 해당 사이즈에서 변경되는 내용만 작성하면 된다. 이미 적용되어 있는 값은 따로 적을 필요 없음

    작성시 주의사항은 CSS에서 미디어 쿼리의 작성 위치도 중요하다

    breakpoints를 높은 값에서 낮은 값 순으로 내려가면서 작성해야하며

    순서를 어기게 되면 미디어쿼리 동작이 원활하게 이루어 지지 않을 수 있음



    breakpoints를 생각해서 쿼리를 작성해야함

  • 연산자

    연산자 설명
    and 앞의 소스처럼 조건을 계속 추가할 수 있습니다.
    ,(쉼표) 동일한 스타일 유형을 사용할 미디어의 유형과 조건이 있다면 쉼표를 이용하여 추가합니다.
    only 미디어 쿼리를 지원하는 웹 브라우저에서만 조건을 인식하게 합니다. 이 키워드를 사용하면 미디어 쿼리를 지원하지 않는 웹 브라우저에서는 미디어 쿼리를 무시하고 실행하지 않습니다.
    not not 다음에 지정하는 미디어 유형을 제외합니다. 예를 들어 'not tv'라고 지정한다면 TVz를 제외한 나머지 미디어 유형에만 적용됩니다.
